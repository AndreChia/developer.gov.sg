Overview

During this RELEASE phase, you will:

  • Finalise system and user documentation - Documentations is treated like any other requirement where it should be costed, prioritised and created when stakeholders are willing to invest in it.
  • Training - We train end users, operations staff and support staff to work effectively with our system.
  • Deploy the system - System deployment is a complex endeavor which is just as critical aspect of the software development lifecycle.
