Overview

The SingStat Table Builder is an application that allows users to select and build customized data tables, export these tables in different file formats or plot graphs and charts. It is available for free access on the SingStat website.

What does SingStat Table Builder contain?

SingStat Table Builder contains frequently asked statistics from various subjects, such as national accounts, prices, services, manufacturing, international accounts, population and population structure, and households and housing. Monthyl, quarterly and annual data series are available for users to create customised data tables and plot charts.

System Requirements

The SingStat Table Builder runs on popular web browsers such as Internet Explorer, Mozilla Firefox, Google Chrome and Safari. It is best viewed at a minimum screen resolution of 1024 x 728 pixels.
