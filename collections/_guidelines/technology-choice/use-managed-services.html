Commercial cloud services

Unless prohibited, consider using solutions by commercial cloud providers first. Usage of commercial cloud providers is done through the Government Commercial Cloud.

Cloud adoption results in numerous benefits, including: 

  • IT Delivery and Ops

    • Improved IT delivery/deployment (e.g. reduction in development time and/or reduction in day-to-day support and maintenance e.g. using Infra as Code or CI/CD implementation)

  • Flexible infrastructure

    • With cloud services, you can scale up or down your hosting resources on demand.

  • Agility

    • With practically every service offering automation and seconds-only deployment, it is possible to rapidly deploy application updates and react to increased service demand or security threats.

  • Security

    • Cloud service providers offer robust security features and internationally recognized certifications that would be a challenge for anyone organization to deliver on its own.

Government cloud-based services

Various other cloud-based options built by the Government are available, such as NDI for verifying identities digitally, Isomer for the hosting of static sites, Ship for DevOps tooling and Hats for automated mobile and application testing. See the Technologies page for more details.
