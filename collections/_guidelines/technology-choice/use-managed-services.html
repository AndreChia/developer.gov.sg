Commercial cloud services

Unless prohibited, consider using solutions by commercial cloud providers first. Usage of commercial cloud providers is done through the Government Commercial Cloud.

Cloud adoption results in numerous benefits, including: 

  • IT Delivery and Ops

    • Improved IT delivery/deployment (e.g. reduction in development time and/or reduction in day-to-day support and maintenance e.g. using Infra as Code or CI/CD implementation)

  • Flexible infrastructure

    • With cloud services, you can scale up or down your hosting resources on demand.

  • Agility

    • With practically every service offering automation and seconds-only deployment, it is possible to rapidly deploy application updates and react to increased service demand or security threats.

  • Security

    • Cloud service providers offer robust security features and internationally recognized certifications that would be a challenge for any one organization to deliver on its own.

Government managed services

The Government builds and manages a variety of digital services and tools that development teams can leverage on instead of having to build from scratch. Some of these include:

  • NDI for authenticating citizens and companies, as well as providing consent-based access to citizen data

  • Isomer for the hosting and deployment of static sites

  • Nectar, a managed platform as a service for hosting and deploying applications on the intranet

  • Ship, a CI/CD and team collaboration platform

  • Hats, a functional testing platform for web and mobile applications

  • SGDS, a design system for government websites

For a more comprehensive list, visit the technologies page.
